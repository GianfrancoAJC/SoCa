¡Bienvenido a SoCa!

El FOMO es una patología psicológica descrita como «una aprensión generalizada de que otros podrían estar teniendo experiencias gratificantes de las cuales uno está ausente». Este tipo de ansiedad socialse caracteriza por «un deseo de estar continuamente conectado con lo que otros están haciendo. SoCa es una aplicación web diseñada para acompañarte en este proceso de determinar y tratar el FOMO. ¡Termina con el FOMO en tu vida y se la mejor versión de ti mismo!
